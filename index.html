My Main Power

HTML

HyperText Markup Language.
HTML은 웹 페이지를 만들때 필요한 필수적인 Markup Language입니다. Tag로 정보를 나누면서 정보의 가치를 향상시킵니다.

CSS

Cascading Style Sheet.
CSS는 웹 개발에서 미적인 부분을 담당합니다. CSS의 주요 역할은 웹페이지를 아름답게 꾸며주는 역할을 합니다. Tag의 selector, class, id를 이용해 스타일을 지정할 수 있습니다.

Javascript

Javascript는 객체 기반의 스크립트 프로그래밍 언어로 주로 웹페이지 동작을 동적으로 제어하는 역할을 합니다. 또한 node.js와 같은 런타임 환경과 같이 server-side 네크워크 프로그래밍에도 사용되고 있습니다.

Main Library

JQuery

JQuery는 Javascript library 중 하나이며 오늘날 가장 인기있는 library 중 하나입니다.
JQuery의 문법은 코드 보기, 문서 객체 모델 찾기, 애니메이션 만들기, 이벤트 제어, Ajax 개발을 쉽게 할 수 있도록 디자인되었으며, jQuery는 plug-in을 개발할 수 있는 기능을 제공합니다.
